School Management System

Code On GitHub

This School Management System is a comprehensive platform built using Laravel for the backend, React for the frontend, and MySQL as the database. It is designed to streamline various administrative, academic, and communication tasks in schools. The system facilitates role-based access for admins, teachers, students, and parents. Key features include:

  • Admin Panel: Admins can manage users (teachers, students, parents), assign roles, and perform CRUD operations on classes, subjects, timetables, and fees.
  • Teacher Module: Teachers can manage class attendance, assign homework, schedule exams, and register student marks. They can also send messages to students and parents.
  • Student and Parent Module: Students and parents can view assigned homework, check timetables, exam schedules, and fee statuses, and receive messages from teachers and admins.
  • Database Relations: The system uses a relational database to maintain relationships between students, classes, teachers, and other entities, ensuring smooth data retrieval and operations.
  • Secure Authentication: The system uses Laravel-Sanctum authentication to provide secure access for all roles.

This system simplifies school management, enhances communication, and provides an efficient way to manage academic and administrative tasks in educational institutions.

Pharmacy Management System

Code On GitHub

This Pharmacy Management System is a web-based application built using Laravel (backend), React (frontend), and MySQL (database). It is designed to streamline pharmacy operations by managing inventory, processing sales, handling prescriptions, and maintaining supplier relationships. The system ensures efficient workflow automation while providing secure user access and role-based functionality, making it easier to manage day-to-day tasks. Key features include:

  • Inventory Management: Track medicines, their quantities, expiration dates, and suppliers. Automatically update stock levels upon sales.
  • Supplier Management: Manage a list of suppliers, their details, and the medicines they supply. This might involve a many-to-many relationship between medicines and suppliers.
  • Sales and Billing: Handle customer sales, generate invoices, and manage payments. You likely implemented functionality to calculate totals, apply taxes/discounts, and maintain transaction records.
  • Prescription Management: Allow pharmacists to verify prescriptions and dispense medications accordingly.
  • Secure Authentication: The system uses Laravel-Sanctum authentication to provide secure access for Admin.

JS Shopping Cart

Live Preview Code On GitHub

The JavaScript Shopping Cart is a dynamic client-side application that allows users to add, update, and remove items from their cart seamlessly. Built using JavaScript , it provides real-time updates without requiring page reloads. Key features include:

  • Add to Cart: Users can select products and add them to their shopping cart with a single click.
  • View Cart: A dedicated section displays the current items in the cart, including quantities and total prices.
  • Update Quantities: Users can modify item quantities directly in the cart, with instant recalculations of totals.
  • Remove Items: Easily remove items from the cart, reflecting changes immediately.
  • Persistent State: Cart data stored in local storage to retain items even if the user navigates away from the page.

This shopping cart enhances the user experience by providing a smooth, interactive way to manage products before proceeding to checkout.

Hangman Game

Live Preview Code On GitHub

The Hangman Game is an interactive web-based application created using HTML, CSS, and JavaScript. This classic word-guessing game challenges players to guess a hidden word by suggesting letters within a limited number of attempts. Key features include:

  • Word Selection: The game randomly selects a word from a predefined list.
  • User Input: Players can input letters to guess the hidden word, with real-time feedback on correct and incorrect guesses.
  • Visual Feedback: The game visually represents the hangman's progress, showing incorrect guesses and the corresponding number of attempts remaining.

This Hangman game combines engaging gameplay with a user-friendly interface, providing an enjoyable experience for players of all ages.
